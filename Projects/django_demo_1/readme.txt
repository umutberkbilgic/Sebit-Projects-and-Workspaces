Django Python web app that demonstrates use of Google Cloud Datastore and gQuery.

#########################

To run it;
	1) open a console at "/Desktop/Projects/django_demo_1/mysite"
	2) run the command "python manage.py runserver"
	3) Go to "localhost:8000" and interact.

#########################
	
The comments are extracted from GCloud Datastore. Comments can be added
	to the database from either the Google Cloud Console or the 
	datastore_demo_gui_1.
