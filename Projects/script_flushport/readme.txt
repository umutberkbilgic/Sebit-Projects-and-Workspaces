A custom Python script that stops process and listeners on a given localhost:PORT. Generally that port will
	be 8080 or 8000 (for example gcloud -> 8080 and 8000 (admin), django -> 8000).

	Also can deploy a new app to localhost given the app.yaml file.

###################################

To run it;
	1) Open a console at "~/Desktop/Projects/script_flushport"
	2) Run the command "python flushport.py"
	3) Enter the port number to be flushed.
