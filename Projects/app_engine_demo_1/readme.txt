Google CLoud App Engine deployment (also incooperates Google Cloud Functions)

Simple HTML + JS demo. Prints the result of a GCloud function on the console as well,
	showing that GCloud Functions can be used with App Engine deployments using
	python or nodejs. 

#####################################

To start, 
	1) Open a console at "~/Desktop/Projects/app_engine_demo_1/demo/"
	2) Run the command "dev_appserver.py app.yaml"

	3) After some time, the local development server will launch and the application 
		can be tested at localhost:8080

#####################################

app.yaml is a configuration file.
