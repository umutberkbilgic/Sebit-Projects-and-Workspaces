GUI application using Python and Tkinter that allows users to register / login / comment / view comments / logout

Implementation is modular in the sense that it would be easy t carry it over to a Nodejs or html+js environment to deploy 
	at Google Cloud App Engine or a webserver within a VM at Google Cloud Compute Engine.

#######################################

To run it;
	1) Open a console at "~/Desktop/Projects/datastore_demo_gui_1"
	2) Run the command: "python datastore_gui_demo_1.py"

#######################################

A list of all the users and commands can be seen at Google Cloud web console.
